# jhammant/boothub-profile

Bootstrap any AI agent with my standard setup:

```text
"Hey Claude, fetch boothub.dev/jhammant/swarm and bootstrap me."
```

This is a [boothub](https://boothub.dev) profile. It's a regular GitHub repo — fork it, edit it, point your agent at your fork.

## Presets

| URL | What you get |
|---|---|
| `boothub.dev/jhammant` | Default: skills + commands + project context |
| `boothub.dev/jhammant/lite` | Just a couple of slash commands |
| `boothub.dev/jhammant/swarm` | Default + GitHub-based swarm (zero infra) |
| `boothub.dev/jhammant/swarm-pro` | Default + hosted swarm (boothub.dev backed) + claw-bridge + claude-history MCPs |
